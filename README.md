# Cibersegura | Cybersecurity Awareness Website 🛡️

An educational website designed to raise awareness about common cybersecurity risks and how to stay safe online.

The project presents essential cybersecurity concepts in a simple and accessible format for everyday internet users.

---

## 🌐 Live Demo

👉 https://cibersegura.vercel.app

---

## 📌 Website Sections

The site includes educational sections about:

* Phishing attacks
* Creating strong passwords
* Common online scams
* Basic cybersecurity best practices

---

## ⚙️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 🎯 Purpose of the Project

The objective of this project is to promote **cybersecurity awareness** and help users understand basic digital safety practices.

It was also developed as part of a personal portfolio exploring **security-related web projects**.

---

Cibersegura/

index.html  (HTML, CSS and JavaScript in a single file)

For simplicity and easy deployment, the entire project is implemented in a single HTML file that includes embedded CSS and JavaScript.

This approach makes the application lightweight and easy to host as a static webpage.
